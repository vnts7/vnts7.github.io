State Management Quiz

  1. What is the main value of Java Server Pages?

    It separate display from processing, i.e., separate html from java

  2. How are JSP pages related to servlets?

    JSP page will be compiled into a servlet

  3. How are JSP pages related to HTML?

    Main content of JSP is HTML code

  4. Give an example of a JSP scriptlet and show how it will look in the JSP servlet .

    <% count = count + 1; %>

    This code will run inside service method

  5. Give an example of a JSP declaration and show how it will look in the JSP servlet.

    <%! public void jspInit() { //your initialization code} %>

    This code will only perform once and help to overide jspInit function

  6. Give an example of a JSP comment and show how it will look in the JSP servlet.

    <%-- jsp comment --%>

    It's a comment, not showing when runing

  7. Give an example of a JSP expression and show how it will look in the JSP servlet.

    <%= ++count %>

    will becomes out.print( ++count ); in service method

  8. Give an example of a JSP directive and show how it will look in the JSP servlet.

    <%@ page import="java.util.Date" %>

    This will import java.util.Date package for later use

  9. Explain how an EL expression is evaluated.

    ${firstThing} if firstThing is not an implicit EL object, then search page, request, session and application scopes until attribute "firstThing" is found

    ${firstThing.secondThing}: if firstThing is a bean then secondThing is a property of the bean, else if firstThing is a map then secondThing is a key of the map

  10. Explain how servlet attributes are involved in EL expressions.

    EL expressions will search servlet attributes (from page to request, session and application scopes) until the attribute is found

  11. Explain how servlets and JSPs use request dispatch to interact.

    Have a single controller servlet that takes requests

    Gets request parameters

    Converts and validates them

    Calls object with business logic to do processing

    Forwards results to jsp page for display
